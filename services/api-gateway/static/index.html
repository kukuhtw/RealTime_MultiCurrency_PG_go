Cara Testing (Ringkas)

  1. Jalankan stack: make dev UP
  2. Healthcheck: curl -s http://localhost:8081/healthz dst.
  3. Generate dummy CSV: make dummy-docker N=1000 → tests/data/dummy_transactions.csv
  4. Integration test: make test-integration-docker (cek CSV terbaca)
  5. Load ringan (opsional): kirim banyak POST /payments dan lihat panel di Grafana.
Saat mengirim dari browser lintas port bisa terblokir CORS, jadi gunakan cURL/CLI.
